Date format validation

Write a program using regular expressions that will check if the input date is valid. A valid date in this case is an actual date in the format DD/MM/YYYY.

Note that a day is a digit from 01 to 31; a month is a digit from 01 to 12.


Sample Input:
01/12/2018

Sample Output:
True



Memory limit: 256 MB
Time limit: 15 seconds
